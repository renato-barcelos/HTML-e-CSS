fazendo sozinho

  • ID " # "

    ID serve para identificar um único elemento, não pode ser usado mais de um ID de mesmo nome.

  • Class " . "

    CLass serve para formatar um grupo de elementos, uma classe de elementos, vários elementos podem ter o mesmo nome de class, todos os que estiverem o mesmo nome, terão a mesmo stylo de formatação.

  • Pseudo-class " : "

    serve para dar configurações extras nas class, como por exemplo HOVER, que serve para dar uma regra de stylo ao passar o mouse em cima daquela class (ou ID) que também funciona

  • Pseudo-elementos " :: "

    serve para dar configurações extras para os elementos, como por exemplo o Before e after, que são para adicionar algo antes do elemento ou depois do elemento, respectivamente, ou ambos.

  • Clildren " > "

    serve para configurar algo que está dentro de outro, por exemplo uma Div que tem um parágrafo dentro dela, o Clildren configura o stylo daquele parágrafo sem interferir na DIV
