parágrafos e quebra de linhas

voce pode escrever um parágrafo de qualquer jeito, basta colocar tudo no meio do par de tags p e /p

se precisar quebrar um texto específico como esse, (br)
pode usar a tag br

para fazer um comentário no código sem que seja entendido como programação, é preciso digitar < !-- (sem espaço) para iniciar e o mesmo (sem exclamação) para fechar

para criar marca registrada, digitar ®

para gerar simbolo de copy, digitar © <©>

™ € £ ¥ ¢ Δ

vamos adicionar alguns emogis

para inserir símbolos, sempre começa com &#x para dizer que o que vem é um hexadecimal
